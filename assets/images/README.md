## How to Extract Images from PDF

### Method 1: Using Adobe Acrobat
1. Open the PDF in Adobe Acrobat
2. Right-click on each image
3. Select "Save Image As..."
4. Save with the corresponding filename

### Method 2: Using Preview (Mac)
1. Open the PDF in Preview
2. Select the image using the selection tool
3. Copy and paste into a new file
4. Export as PNG

### Method 3: Using Online Tools
1. Use tools like pdf2png.com or smallpdf.com
2. Convert each page to PNG
3. Crop the images as needed

### Method 4: Take Screenshots
1. Open the PDF at a good zoom level
2. Use screenshot tool to capture each figure
3. Crop to remove excess white space
4. Save with the corresponding filename

## After Adding Images

Once all images are added to this directory:
1. Commit the changes to your repository
2. Push to GitHub
3. The images will automatically display on your GitHub Pages site
